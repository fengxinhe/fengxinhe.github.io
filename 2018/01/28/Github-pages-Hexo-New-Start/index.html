Github pages + Hexo = New Start

Posted on 2018-01-28 | In Tech article | | View

刚刚用Github Pages搭建了新的个人博客，又是一个新的开始。以前在CSDN上断断续续写过一点东西，但是总没有坚持下去。 之前还注册过一个域名，竟然没用过就过期了。这次花了一天的时间总算有了一块自己的地盘。希望自己能坚持下去。

不得不赞叹github是一个极为优秀的平台，在为编程人员不断尽力提供开源服务。Github Pages是一个托管静态网页的平台，可以绑定自己的域名，最重要的是免费！创建自己的网站很简单，按照GithubPages官网的步骤就好。

为了节约时间，你需要一个博客生成器，Jekyll和Hexo都是很好的选择。他们都可以直接使用Markdown来编辑文章。Hexo有中文支持，需要node，而jekyll需要ruby。我因为当时安装不上jekyll而改用hexo。这是因为一个Linux系统的bug导致的，这个悲伤的故事我会在下一篇文章里总结。
